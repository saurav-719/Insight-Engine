# 📊 Insight Engine

A powerful end-to-end data analysis tool built with Streamlit.

## Features
- 📁 Upload CSV/Excel datasets
- 🔍 Auto EDA — statistics, correlations, missing values
- 📊 7 interactive chart types
- 🧹 Data cleaning — nulls, duplicates, outliers, encoding, scaling
- 🤖 AI-powered insights via NVIDIA API
- 📄 PDF report export

## Setup
pip install -r requirements.txt
streamlit run app.py

## Requirements
- Python 3.9+
- NVIDIA API key (for AI Insights)
